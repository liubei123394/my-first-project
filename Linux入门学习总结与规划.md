# Linux入门学习总结与规划

### 今天掌握的核心技能：

1. **文件系统操作**：`mkdir`（创建目录）、`cd`（切换目录）、`ls -l`（查看文件详情）

2. **文件操作**：`echo >`（创建并写入文件）、`cat`（查看文件内容）、`cp`（复制文件）、`rm`（删除文件）。

3. **软件管理**：`sudo apt update`（更新软件源）、`sudo apt install`（安装软件）。

4. **文本编辑**：使用 `nano` 编辑器修改文件并保存。

5. **目录可视化**：使用 `tree` 工具直观查看目录结构。
