# markdown笔记

## 1. Markdown 是什么？

**Markdown 是一种 *用简单符号排版*  的文本格式**

- 不是软件

- 不是代码

- 就是**纯文本 + 几个符号**

- 后缀是 `.md`

- **MarkText** 就是专门打开、编辑 `.md` 文件的工具。

## 2. 为什么要用 Markdown？

### 1.干净

没有花里胡哨的按钮，专心写内容。

### 2.格式超级简单

不用点鼠标选 “标题 1、标题 2、加粗”

只打符号：

- `#` 标题
- `**` 加粗

### 3.用于写笔记、作业、代码笔记

**适合：**

- 课堂笔记
- 高数公式
- C 语言代码
- 复习提纲

### 4.到处能用

- 微信公众号
- 知乎
- 掘金
- GitHub
- 各种笔记软件。

### 5.比 Word 轻、快、不乱

Word 经常：

- 排版乱飞

- 图片乱跑

- 文件巨大
  
  > Markdown **永远不乱**。

## 3. 怎么样用

### 1, 标题

markdown

```
# 一级标题
## 二级标题
### 三级标题
```

### 2,加粗

markdown

```
**重点内容**
```

### 3.列表

markdown

```
- 知识点1
- 知识点2
- 知识点3
```

### 4,代码

markdown

```
```c
int a = 10;
```
